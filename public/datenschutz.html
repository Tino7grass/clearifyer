2. Erhebung und Verarbeitung von Daten

Clearifyer ist ein Adressprüfungsdienst für Blockchain-Adressen. Bei der Nutzung des Tools werden folgende Daten verarbeitet:

  • Blockchain-Adressen: Die eingegebene Adresse wird zur Prüfung an externe Datenquellen (Etherscan, Chainabuse, OFAC-Listen) übermittelt. Es handelt sich um öffentliche Blockchain-Daten ohne Personenbezug.
  • Kontextangaben: Die Antwort auf die Kontextfrage (Herkunft der Adresse) wird ausschließlich für die Risikobewertung verwendet und nicht gespeichert.
  • Server-Logs: Beim Aufruf der Website werden technische Daten (IP-Adresse, Zeitstempel, Browser) im Rahmen des Netlify-Hostings temporär gespeichert.

3. Keine Nutzerkonten, keine Cookies

Clearifyer erfordert keine Registrierung und setzt keine Tracking-Cookies ein. Es werden keine personenbezogenen Daten dauerhaft gespeichert.

4. Externe Datenquellen

Zur Adressprüfung werden folgende externe Dienste genutzt:

  • Etherscan (etherscan.io) — Blockchain-Daten Ethereum
  • Chainabuse (chainabuse.com) — Betrugsreports
  • OFAC SDN List — US-Sanktionsliste (US Treasury)
  • EU-Finanzsanktionsliste — Europäische Sanktionsdaten
  • Anthropic Claude API — KI-gestützte Risikobewertung

Die übermittelten Daten beschränken sich auf die geprüfte Blockchain-Adresse. Es werden keine personenbezogenen Daten an diese Dienste weitergegeben.
